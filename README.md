# kwork-monitor-bot

Бот мониторит биржу Kwork и присылает уведомления о новых заказах по заданным ключевым словам и категориям. Делал для себя, чтобы не пропускать подходящие проекты.

## Что умеет

- автопроверка каждые 5 минут
- фильтр по ключевым словам и категориям (настраивается в боте)
- уведомление только о новых заказах — уже просмотренные не повторяются
- команда `/all` — все подходящие заказы за сессию
- heartbeat раз в 2 часа — сообщение что мониторинг живой
- `/settings` — управление ключевыми словами и категориями

## Требования

Python 3.10+

## Стек

aiogram 3, aiohttp, aiosqlite, beautifulsoup4, python-dotenv

## Запуск

```
pip install -r requirements.txt
cp .env.example .env
# вставить BOT_TOKEN
python main.py
```
